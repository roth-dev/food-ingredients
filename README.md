## React Native Open source project

Installing

```bash
$ expo install
# or
$ npm install
```

## Getting Started

First, run the expo development server:

```bash
$ expo start
# or
$ yarn start
# or
$ yarn start ios
# or
$ yarn start android
```

expo SDK 40

Publish to expo server

```bash

$ expo publish

```

## Build

IOS

```bash

expo build:ios

```

ANDROID

```bash

expo build:android

```
